# Conjuntos en Python

La teor√≠a de conjuntos es una rama de las matem√°ticas y de la l√≥gica que estudia objetos llamados *conjuntos*, formados por elementos que, al agruparse, adquieren propiedades propias. Su finalidad es analizar las caracter√≠sticas de estos conjuntos y las operaciones que se pueden realizar entre ellos.

## ¬øQu√© es un Conjunto?

Un conjunto es una colecci√≥n de elementos considerada en s√≠ misma como un objeto. Por ejemplo:

- **Conjunto de las vocales:**  
  V = {a, e, i, o, u}
- **Conjunto de los d√≠as de la semana:**  
  D = {domingo, lunes, martes, mi√©rcoles, jueves, viernes, s√°bado}

## Operaciones con Conjuntos

La teor√≠a de conjuntos nos permite llevar a cabo diversas operaciones que nos ayudan a comparar y combinar colecciones de elementos. A continuaci√≥n, se presentan algunas de las operaciones m√°s importantes:

### Uni√≥n de Conjuntos

La uni√≥n de conjuntos consiste en identificar todos los elementos que pertenecen a uno o a varios conjuntos. Se define como:

‚ÄÉ‚ÄÉA ‚à™ B = {x | x ‚àà A ‚à® x ‚àà B}

![union de conjuntos](https://static.platzi.com/media/articlases/Images/Uni%C3%B3n%20de%20conjuntos.png)

Es decir, re√∫ne todos los elementos de A y B, sin repetir aquellos que se encuentren en ambos.

### Intersecci√≥n de Conjuntos

La intersecci√≥n es la operaci√≥n que obtiene un conjunto formado √∫nicamente por los elementos comunes a ambos conjuntos. Se expresa como:

‚ÄÉ‚ÄÉA ‚à© B = {x | x ‚àà A ‚àß x ‚àà B}

![intersecci√≥n de conjuntos](https://static.platzi.com/media/articlases/Images/Intersecci%C3%B3n%20de%20conjuntos.png)

### Diferencia de Conjuntos

La diferencia entre dos conjuntos genera un nuevo conjunto con los elementos del primero que no se encuentran en el segundo:

‚ÄÉ‚ÄÉA - B = {x | x ‚àà A ‚àß x ‚àâ B}

![Diferencia de conjuntos](https://static.platzi.com/media/articlases/Images/image%28237%29.png)

Es importante destacar que A - B no es lo mismo que B - A.

### Diferencia Sim√©trica

La diferencia sim√©trica se compone de los elementos que est√°n en A pero no en B, junto con aquellos que est√°n en B pero no en A. Se define de la siguiente manera:



$A \triangle B = {x | x \in A-B \lor x \in B-A} = (A-B) \cup (B-A)$


![Diferencia sim√©trica de conjuntos](https://static.platzi.com/media/articlases/Images/Diferencia%20sim%C3%A9trica%20de%20conjuntos.png)

Por ejemplo, si tenemos:

‚ÄÉ‚ÄÉP = {a, e, i, o, u}  
‚ÄÉ‚ÄÉQ = {a, b, c, d, e}

Entonces,  
‚ÄÉ‚ÄÉP Œî Q = {i, o, u, b, c, d}

### Complemento de Conjuntos

El complemento de un conjunto A se refiere a los elementos que pertenecen al conjunto universal U, pero no a A:

‚ÄÉ‚ÄÉA·∂ú = {x | x ‚àâ A ‚àß x ‚àà U}

![Complemento de conjuntos](https://static.platzi.com/media/articlases/Images/Complemento%20de%20conjuntos.png)

Considera el siguiente ejemplo:

- U = {1, 2, 3, 4, 5, 6, 7, 8, 9}
- A = {5, 6, 7, 8, 9}
- B = {1, 6, 7, 8}

Entonces:

- A·∂ú = {1, 2, 3, 4}
- B·∂ú = {2, 3, 4, 5, 9}
- A - B = {5, 9}
- B - A = {1}
- A Œî B = {1, 5, 9}

## Tipos de Scope

# Funciones de Orden Superior

## ¬øQu√© son y c√≥mo funcionan?  
Las funciones de orden superior (HOF) son aquellas que pueden recibir otras funciones como par√°metros o devolverlas como resultado. Esto permite crear soluciones flexibles y modulares.  

---

## Ejemplo B√°sico  
### Funci√≥n cl√°sica + HOF  
<div style="background: #1E1E1E; padding: 10px; border-radius: 8px; width: fit-content; font-family: monospace; color: white;">
  <div style="display: flex; gap: 6px; padding: 5px;">

    <span style="width: 12px; height: 12px; background: #27C93F; border-radius: 50%; display: inline-block;"></span>
  </div>
  <hr style="border: 1px solid black; background: none; margin:0; padding:0;  height: 0px; ">

```python
def incrementar(x):
    return x + 1

def hof(x, funcion):
    return x + funcion(x)

resultado = hof(2, incrementar)
print(resultado)  # Output: 5 (2 + incrementar(2)=3 ‚Üí 2+3=5)
```
</div>
<br>  

---

## Integrando Funciones Lambda  
Las lambdas simplifican el uso de HOF al evitar definir funciones completas:

### 3 Ventajas de Usar Lambdas con HOF

# Funci√≥n Map en Python: Transformaci√≥n Eficiente de Listas  

## ¬øQu√© hace y c√≥mo funciona?  
La funci√≥n `map` aplica una operaci√≥n a cada elemento de una lista, generando una nueva lista con los resultados. Mantiene la misma cantidad de elementos pero transformados.

## Uso B√°sico: Multiplicar Elementos

```python
numeros = [1, 2, 3, 4]

print(resultado)  # Output: [2, 4, 6, 8]
```

# Funci√≥n Reduce en Python: Acumulaci√≥n de Datos Simplificada  

## ¬øQu√© es y para qu√© sirve?  
La funci√≥n `reduce` procesa una lista para generar un √∫nico resultado acumulativo. Ideal para sumatorias, productos m√°ximos, o cualquier operaci√≥n que combine elementos secuencialmente.  

---

## Implementaci√≥n Paso a Paso  
### Requisito: Importar `functools`  
<div style="background: #1E1E1E; padding: 10px; border-radius: 8px; width: fit-content; font-family: monospace; color: white;">
  <div style="display: flex; gap: 6px; padding: 5px;">
    <span style="width: 12px; height: 12px; background: #FF5F57; border-radius: 50%; display: inline-block;"></span>
    <span style="width: 12px; height: 12px; background: #FFBD2E; border-radius: 50%; display: inline-block;"></span>
    <span style="width: 12px; height: 12px; background: #27C93F; border-radius: 50%; display: inline-block;"></span>
  </div>
  <hr style="border: 1px solid black; background: none; margin:0; padding:0;  height: 0px; ">

```python
import functools

numeros = [1, 2, 3, 4]

# Sumar todos los elementos
resultado = functools.reduce(lambda acumulado, item: acumulado + item, numeros)
print(resultado)  # Output: 10
```
</div>
<br>  

---

## Desglose de Iteraciones  
### Visualizando el proceso interno  
<div style="background: #1E1E1E; padding: 10px; border-radius: 8px; width: fit-content; font-family: monospace; color: white;">
  <div style="display: flex; gap: 6px; padding: 5px;">

```python

# Output paso a paso:
```

## 3 Ventajas Clave de Reduce  
1. **C√≥digo conciso**: Reemplaza bucles `for` con una sola l√≠nea  
2. **Flexibilidad**: Funciona con cualquier operaci√≥n acumulativa (suma, multiplicaci√≥n, concatenaci√≥n)  
3. **Legibilidad**: Claridad al expresar la intenci√≥n del c√≥digo  

---

## Soluci√≥n: `if __name__ == "__main__"`  
Este bloque garantiza que el c√≥digo solo se ejecute cuando el archivo es el punto de entrada principal.  

<div style="background: #1E1E1E; padding: 10px; border-radius: 8px; width: fit-content; font-family: monospace; color: white;">
  <div style="display: flex; gap: 6px; padding: 5px;">
    <span style="width: 12px; height: 12px; background: #FF5F57; border-radius: 50%; display: inline-block;"></span>
    <span style="width: 12px; height: 12px; background: #FFBD2E; border-radius: 50%; display: inline-block;"></span>
    <span style="width: 12px; height: 12px; background: #27C93F; border-radius: 50%; display: inline-block;"></span>
  </div>
  <hr style="border: 1px solid black; background: none; margin:0; padding:0;  height: 0px; ">

```python
# main.py (versi√≥n corregida)
def ejecutar_como_script():
    print("Modo script activado ‚úÖ")
    usuario = input("Nombre: ")
    print(f"Hola, {usuario}!")

if __name__ == "__main__":
    ejecutar_como_script()
```
</div>
<br>  

### ¬øC√≥mo funciona?  
- **Ejecuci√≥n directa**: `__name__` es `"__main__"` ‚Üí Se ejecuta el bloque.  
- **Como m√≥dulo**: `__name__` es el nombre del archivo (ej: `"main"`) ‚Üí No se ejecuta.  

---

## Recomendaciones Clave  
1. **Encapsula la l√≥gica principal en funciones**: Facilita el control de ejecuci√≥n.  
2. **Prueba ambos modos**: 

   ```bash
   # Como script
   python main.py

   # Como m√≥dulo (en otro archivo)
   import main
   ```  

3. **Evita c√≥digo global**: Todo lo que no sean definiciones (funciones/clases) debe ir dentro del bloque `if __name__`.
